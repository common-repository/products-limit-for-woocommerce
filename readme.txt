Products limit is a plugin extension of WooCommerce that add minimum and maximum quantity limit to cart and checkout page of your ecommerce. 


== Description ==

This plugin is an extension of WooCommerce, providing minimum and maximum quantity limit to the product of your e-commerce, 
it is useful in the case you want to set a minimum or maximum limits of products in the cart on every shopping session. The limit is calculated considering the sum of every product added to cart.
In case this limit is reached, the extension provides a warning banner in the cart or in the checkout page and disable the checkout process. 
= Features: =
* Set a max. or min. quantity limits of products before checkout.
* Add custom button "Continue Shopping" in the warning banner in cart and checkout page.
* Dashboard widget with settings link and summary of the plugin.


== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. When the plugin is activated the settings are available in the WooCommerce menu > settings > products tab > min/max products limit. 
4. In the settings section you can find the number fields that can be set and save the minimum and maximum quantity of products.
5. To enable only one of the products quantity limit just leave blank the corrispondent field.
6. To enable "Continue Shopping" button in the warning banner check the corrispondent checkbox.
7. When plugin is activated you can find the products limit dashboard widget in the main dashboard of WordPress

== Frequently Asked Questions ==

= How can I disable maximum or minimum quantity limit if I need only one of them? =

Just leave blank the corrispondent field and press the save button.

= Where are the settings of the plugins? =

As an extension of WooCommerce, the settings of the plugin are in the woocommerce menu, settings > products tab > products limit submenu.

= Is Products limit plugin ready for Multisite use? =

This plugin works in Multisite configuration, you can find the settings in the single sites of the network.
